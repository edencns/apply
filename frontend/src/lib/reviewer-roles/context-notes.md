# 담당별 검증 에이전트 — 컨텍스트 노트

작업 중 내린 결정과 근거를 계속 추가. 다음 세션이 재유도 없이 이어받기 위함.

## 확정된 설계 결정 (사용자 합의)

1. **구현 형태**: 앱 런타임 기능 (배포된 웹앱 안에서 동작). Claude Code 에이전트나 코드 모듈이 아님.
2. **책임 범위**: LLM은 「추출 + 조건 대조 + 제안」까지만. 최종 적합·부적합 판정은 사람(담당자)이 클릭. (Codex와 합의한 「LLM은 추출만, 판정은 룰·사람」 원칙)
3. **검증 입력**: Phase 1에서 페이지 분류된 「자기 유형 관련 서류 페이지」만 받음 (전체 묶음 PDF 아님). 비용·정확도 절감.

## 7개 담당 역할

1. **common (청약 공통사항)** — 모든 당첨자 공통. 무조건 먼저 실행. 무주택 세대구성·거주기간·청약통장·소형저가·해외체류 등.
2. **institution (기관추천)** — 특별공급. 추천서·자격확인, 청약통장(국가유공자·장애인 등 제외).
3. **multichild (다자녀)** — 특별공급. 미성년 자녀 3명+·100점 배점표.
4. **newlywed (신혼부부)** — 특별공급. 혼인 7년·소득·자산·자녀 우선순위.
5. **elderly (노부모부양)** — 특별공급. 만65세 직계존속 3년 부양·세대주·84점 가점.
6. **firstlife (생애최초)** — 특별공급. 생애 무주택·5년 소득세·소득 160%·자산.
7. **general (일반공급)** — 특별공급 제외. 청약순위·지역우선·소형저가 예외 인정(일반공급만).

## 실행 흐름

```
모든 당첨자 → ① common 항상 먼저
            → supply_type 분기
               ├ 일반공급 → ⑦ general
               └ 특별공급 → ②~⑥ 중 해당 1명
```

## 지식의 두 층위

- **법령 기반 일반 룰** (고정): 신혼부부 7년, 다자녀 배점표, 노부모 가점표 등. reviewer-roles.ts에 인코딩.
- **공고별 수치** (가변): 소득 한도액·예치금·규제 여부·기준일. announcement.eligibility_rules에서 읽음 (Phase A에서 사용자 검증한 값).

## 사용자 검토 결과 반영 (청약_정리_검토결과.md)

- **공통**: 인감 3개월 기준 삭제 → 「공고일(2022.03.25) 이후 발행분」. 신분증 유효성은 운영검증 분리. 거주 우선은 일반/특공 6개월 vs 1순위우선 1년 별도 필드. 분양권 매수자=매매대금 완납일(미분양 제외 혜택 없음). 공통서류 특공/일반 분리.
- **기관추천**: 복무확인서=장기복무군인 해당자만. 청약통장 면제=국가유공자·보훈·장애인·도시재생부지제공자+철거인. 추천서 핵심=사업주체 통보 명단 포함. 과거 특공당첨 제외.
- **다자녀**: 시도거주 15점=강원도(강릉 아님). 초본 명시. 3세대=직계존속 무주택+3년 동일등본. 한부모=증명서+5년.
- **신혼부부**: 건강보험료 납부확인 삭제(필수는 건강보험자격득실). 추첨/자산 시 부동산소유현황·비사업자각서. 2순위="1순위 아닌 자"(무자녀+2018.12.11 특례). 맞벌이 우선 합산120% AND 1인≤100%, 일반 1인≤140%.
- **노부모**: 만60세+ 직계존속(피부양자·배우자) 유주택=부적격 「필수」. 직계존속 출입국(3년내 90일초과 불가). 부양기간 초본 확인.
- **생애최초**: 1순위 조건 필요(비규제라도 제28조1항+청약통장 6개월+예치금, 투기/청약과열 추가요건만 미적용). 5년 소득세=본인 기준. 0원=납부의무 있으나 공제로 0원만 인정. 1인가구=소득 또는 자산 중 하나.
- **일반공급**: 제출서류=신분증·인감·등본·초본·가족관계(주택서류는 소명단계). 청약통장 순위는 자격검증 항목(제출서류 목록과 구분). 전용85㎡↓ 가점40+추첨60.
